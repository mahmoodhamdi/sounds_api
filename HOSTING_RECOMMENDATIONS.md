## Deployment Guide (دليل النشر)

### STC Cloud Deployment
```bash
# 1. Create VDC with Ubuntu 22.04
# 2. SSH to server
ssh user@your-stc-ip

# 3. Install dependencies
sudo apt update
sudo apt install python3 python3-pip python3-venv nginx

# 4. Clone and setup
git clone https://github.com/your-repo/sounds_api.git
cd sounds_api
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pip install gunicorn

# 5. Run with Gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

### Google Cloud Run Deployment
```bash
# 1. Create Dockerfile
cat > Dockerfile << 'EOF'
FROM python:3.11-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt gunicorn
COPY . .
CMD exec gunicorn -b :$PORT -w 4 app:app
EOF

# 2. Build and deploy
gcloud builds submit --tag gcr.io/PROJECT_ID/sounds-api
gcloud run deploy sounds-api --image gcr.io/PROJECT_ID/sounds-api --region me-central2
```
